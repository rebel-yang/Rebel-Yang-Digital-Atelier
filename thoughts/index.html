THOUGHTS

Phase Question Actions Output
第一阶段：
主题定义与规划		 我要做个关于“什么”的作品？ 需求分析 明确创作的核心概念、情绪或想讲述的故事。
市场定位 思考作品的受众与想达成的体验。 		1. 核心主题
第二阶段：
生成前规划		 它应该“长什么样”、“感觉如何”？ 风格定义 结合偏好（极简/极繁），确定视觉风格与主色调。
结构化拆解 将主题拆解为连贯的“分镜头”或“视觉序列”。 		2. 视觉风格指南
3. 主Prompt（定调）
4. 分镜头脚本
第三阶段：
生成与粗剪		 如何高效产出并组装素材？ 敏捷开发 批量生成备选镜头，快速验证效果。
MVP构建 筛选最佳素材，拼接出故事主线草稿。 		5. 原始视频素材库
6. 粗剪版本
第四阶段：
后期与定稿		 如何提升至发布品质？ 核心优化 精剪节奏、统一色调、添加动效/粒子。
集成发布 根据“先听后感”原则，完成音效设计、混音，最终渲染。 		7. 成片
Phase 1:
Theme Definition		 What is my work “about”? Requirement Analysis Define the core concept, emotion, or story.
Market Positioning Consider the target audience and desired experience. 		1. Core Theme

Pre-generation		 What should it “look like”? Style Definition Determine visual style & primary color palette.

Generation		 How to produce efficiently? Agile Development Batch generate candidate shots.

Phase 4:
Finalization		 How to refine? Core Optimization Pacing, color grade, effects.
Integration Sound design & mixing, final render. 		7. Final Video
